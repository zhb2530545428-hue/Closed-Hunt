## 四、如何确认

1. `npm run dev` 后打开 `/dev/map-checker`。
2. 顶部「图结构自检」应显示通过。
3. 点选起点/终点，核对「最短路径 / 步数 / 经过房间 / 是否过激光室 / 特殊移动」是否符合实图直觉。
4. 看「起点的直接相邻」列表，逐房间对照图片确认相邻关系。
5. 发现错误 → 改 `mapGraph.ts` 三个数组 → `npm test` 复测。

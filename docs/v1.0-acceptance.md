# 《禁闭逃杀》v1.0 验收脚本

本文件指导从 0 开始验证一局，覆盖 v1.0 验收标准（开发文档第 15 节）。

可在两种模式下验收：
- **本地模式**（零配置）：一台电脑开多个浏览器标签页热座模拟 9 人，适合快速跑通规则。
- **远程多人模式**（配置 Supabase 后）：多设备 / 多浏览器真实联机，适合验证同步与断线重连。

---

## A. 环境准备

### A1. 本地模式
```bash
npm install
npm run dev
```
打开 http://localhost:3000 ，首页底部应显示「本地模式」。

### A2. 远程多人模式
1. 按 README「配置 Supabase」执行 `supabase/schema.sql`，填好 `.env.local` 三个变量。
2. `npm run dev`，首页底部应显示「真实多人模式」。

---

## B. 开局流程验收

1. 首页填房主昵称 →（本地热座可勾选「开发调试模式」便于少人测试）→ **创建房间**，记下 6 位房间码。
2. 大厅点 **复制邀请链接**；远程模式下用其他浏览器/手机打开该链接。
3. 在 9 个座位分别填昵称 **加入**（本地模式可在同一屏填满 9 个；远程模式每台设备占 1 座）。
4. 每名玩家设置：**职业**（可随机）、**基因点**（武力+速度+负重=10，未满足无法准备）、**出生房间**，然后 **准备**。
5. 房主在「房主控制」点 **开始游戏**（正式局需 9 人全准备；调试模式允许少人）。
6. 验证：进入第 1 轮自由阶段；公开日志出现「游戏开始」「富豪获得金条」等设置日志。

---

## C. 自由阶段 / 交易验收

1. 打开任一玩家「我的面板」（`/room/码/play`），确认显示昵称/职业/生命/状态/基因/负重/道具/位置/顺位卡。
2. 在「自由阶段交易」选择交易对象 + 给出 1 张道具或顺位卡 → **发起交易**。
3. 切到对方身份（本地热座用顶部身份切换；远程模式即对方设备），在「收到的交易请求」**接受**。
4. 验证：双方道具/顺位卡已转移；公开日志记录该笔交易；若接受方超重会出现提示。
5. 房主控制台「待处理交易」可 **取消** 未处理的交易。

---

## D. 行动阶段验收

1. 房主控制台 → **进入行动阶段**（顺位卡已在自由阶段抽取）。
2. 玩家面板「本轮行动」：
   - 地图绿色高亮可达房间，点击选择目标，查看 **路径 / 步数 / 特殊移动 / 激光警告**；
   - 验证：不能选原地（必须移动）、不能选超速房间；
   - 存活玩家选 **毒气投票楼层**；
   - 有职业主动技能者（化学家/催眠师/预言家/慈善家）填写技能；
   - 到达抽卡房间可 **抽卡**；超重出现 **丢弃面板**；
   - 点 **确认移动并提交本轮行动**。
3. 验证经过型效果：让一名存活玩家路径经过 `102 激光室`，提交后立即 -1 生命并写日志。
4. 房主控制台显示 **提交进度 X/9**、毒气投票统计、火箭筒目标参考。

---

## E. 结算验收

1. 全员提交后，房主 → **进入结算阶段** → **生成结算预览**。
2. 验证预览按固定顺序逐步展示：房间效果 → 战斗 → 暗影 → 火箭筒 → 毒气 → 水粮 → 道具回血/状态 → 死亡/复活，并显示每名玩家生命变化。
3. 点 **确认应用结算**：结果写入状态与公开日志，自动进入下一轮（或第 6 轮后最终结算）。
4. 重点规则抽查（可借助房主纠错构造场景）：
   - 同房 2 人战斗按战力差扣血；3 人乱斗；持枪压制无枪玩家；
   - 暗影与存活同房吸血，累计 ≥2 下一轮复活；暗影使者免疫且回血；
   - 毒气按轮次伤害，防毒面具/停机坪/复活当轮免疫；化学家修正生效；
   - 第 2 轮起未交水粮扣血，餐厅 B204 免除；
   - 生命归 0 变暗影，遗物进 B701，入殓师负重 +1 并随机获得 1 张。

---

## F. 房主纠错 / 快照 / 回滚 / 导出导入

1. 房主控制台「手动修正玩家」：±1 生命、设暗影/存活、重置提交。
2. 「高级纠错工具」：切换毒气楼层、改位置/基因/顺位/技能次数、增减道具。
3. 「阶段快照 / 回滚」→ 刷新列表 → 选最近快照 **回滚到此**，验证状态回退。
4. 「导出 / 导入房间状态」→ 导出 JSON（自动复制）→ 修改/粘贴 → **导入并覆盖**，验证生效。

---

## G. 断线重连验收（远程模式）

1. 任一玩家在游戏中 **刷新页面**：应自动回到自己的座位与最新状态，已提交行动不丢失。
2. **关闭再打开** 同一邀请链接（同浏览器）：仍恢复身份。
3. 房主刷新：仍为房主，控制台可继续操作。

---

## H. 终局验收

1. 跑到第 6 轮结算确认后进入 `GAME_OVER`；或构造「仅剩 1 名存活」提前结束；或全员暗影结束。
2. 验证：存活玩家金条自动兑换生命（不超上限）。
3. 公共战况页底部 **最终排名（金魔方积分）**：存活优先 → 生命 → 武力；暗影按生前武力；全员暗影按生前上一轮生命；第 1 名得人数分（9 人局 9 分）依次递减。
4. 复盘：排名 + 公开日志（含每轮毒气、死亡/复活、关键事件）+ 房主可导出房间 JSON 作完整复盘。

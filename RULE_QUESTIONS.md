# 规则待确认清单（RULE_QUESTIONS）— v1.0.6

> 本文件只保留当前仍未确认、且不应由实现自行编造的规则问题。
> v1.0.6 已明确的问题已从旧清单删除。

---

## Q1. 慈善家赠予导致「短暂超过负重」后的复检时机

- **当前实现**：慈善家在结算阶段赠出 1 张具体道具后，目标玩家可以短暂超过负重；本轮结算阶段不强制其立刻丢弃。
- **规则疑点**：规则手册只写「可短暂超过负重」，未说明何时重新检查并要求丢弃。
- **当前取舍**：保留到下一轮自由阶段负重检查时处理。
- **待确认**：是否需要在结算结束后立即让目标丢弃至不超重，还是继续沿用下一轮自由阶段复检？

## Q2. 公开「是否上交水粮」与隐藏位置之间的取舍

- **规则手册**：8.3 将「玩家是否上交水粮」列为公开信息。
- **既有实现约束**：为了避免通过水粮扣血、餐厅免除反推位置，逐人水粮结果只进入本人私密日志和房主裁判日志；公开日志只在最后统一公开生命值。
- **当前取舍**：不公开逐人是否上交水粮。
- **待确认**：是否需要公开「某角色上交/未上交水粮」但仍不公开扣血原因和餐厅免除？

## Q3. 远程模式是否需要服务端级私密字段裁剪

- **当前实现**：本地热座和房主裁判视图保留完整对局状态；普通玩家主要通过 UI 和日志隐藏不该看的信息。
- **风险**：如果未来用于不可信远程玩家，客户端仍可能通过调试工具看到完整房间状态、位置、手牌或裁判信息。
- **待确认**：是否要在后续版本做服务端按身份返回裁剪快照的架构改造？

## v1.0.6 已删除的历史待确认项

以下问题已由 v1.0.6 明确，不再列为待确认：

- 黑客关闭房间按行动阶段即时效果 / 结算阶段效果区分。
- 饮品师多瓶果汁同一目标不可被使用多瓶。
- 私家侦探跟踪到抽卡房间后按剩余库存抽卡。
- 被催眠者强制进入房间后可正常触发终点房间效果和抽卡。
- 暗影使者按暗影实际吸到的生命点数恢复。
- 预言家没有决定顺位能力，也没有意见领袖的额外毒气票权。
- 顺位卡交易只允许交换，不支持单向赠予。
- 慈善家速度为 1 的极端基因转移问题保持现状。
- 慈善家后续死亡 / 变暗影时，挂起的基因收益仍然有效。
- 首轮出生战斗后剩余 0 或 1 名存活玩家时继续沿用提前结束规则。
- 被催眠目标在行动前死亡 / 变暗影的问题按正常流程不成立处理。
- 催眠失败消耗次数，但不标记目标已被催眠。
- 操作室不引入基础基因 / 永久加成拆分，保持当前总和重分配。
- 慈善家赠予事件公开赠予双方角色名，不公开房间、路径或具体道具。
